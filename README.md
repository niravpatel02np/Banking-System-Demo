# Banking System (Python)

A Python-based **Banking Management System** to manage customers, accounts, and transactions using SQLite. This project demonstrates basic CRUD operations, transaction handling, and report generation in a console-based application.

---

## Features

- Add, update, and delete customers  
- Open, view, and manage bank accounts  
- Perform transactions: deposit, withdraw, and transfer  
- Generate transaction history and account reports  
- SQLite database for data persistence  
- Import initial data from Excel
